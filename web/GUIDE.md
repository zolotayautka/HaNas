# HaNas Web 클라이언트 실행 가이드

## 실행 방법

1. **서버 실행** (먼저 실행 필요)
   ```bash
   cd server
   go run .
   ```

2. **웹 클라이언트 실행**
   ```bash
   cd web
   npm run dev
   ```

3. 브라우저에서 http://localhost:3000 접속

## 주요 기능

### 로그인/회원가입
- iOS 스타일의 그라데이션 배경
- 부드러운 애니메이션과 전환 효과
- 입력 필드 포커스 시 하이라이트

### 파일 관리
- **그리드 레이아웃**: 파일과 폴더를 아이콘으로 표시
- **파일 업로드**: 버튼 클릭으로 여러 파일 업로드
- **폴더 생성**: 새 폴더 만들기
- **복사/잘라내기/붙여넣기**: 클립보드 기능
- **이름 변경**: 파일/폴더 이름 수정
- **삭제**: 파일/폴더 삭제
- **공유 링크**: 파일 공유 URL 생성 및 복사
- **다운로드**: 파일 다운로드

### 다중 선택
- 선택 모드 활성화
- 여러 항목 선택
- 선택된 항목 일괄 복사/잘라내기/삭제

### UI 특징
- iOS/iPadOS 스타일의 디자인
- 부드러운 애니메이션
- 반응형 레이아웃 (모바일/태블릿/데스크톱)
- 다크 텍스트와 밝은 배경
- 파일 타입별 아이콘

## 화면 구성

### 로그인 화면
- 앱 아이콘 및 제목
- 사용자명/비밀번호 입력
- 로그인/회원가입 전환

### 파일 목록 화면
- 상단 헤더 (경로, 홈 버튼, 상위 폴더 버튼, 선택 모드)
- 액션 버튼 (업로드, 새 폴더, 붙여넣기, 새로고침)
- 파일 그리드
- 컨텍스트 메뉴 (우클릭/메뉴 버튼)

## 개발 정보

- React 18
- Vite (빠른 개발 서버)
- iOS 스타일 CSS
- Axios (API 통신)
- React Router (라우팅)
